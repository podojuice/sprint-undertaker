Claude Code Hook

Install the local collector, then add one hook snippet to Claude Code.

Idle RPG tracks Claude Code turns from UserPromptSubmit to Stop and only sends privacy-safe summary metrics.

1. Create an installation

Open App, log in, and create a Claude Code installation. That gives you a dedicated API key and install command.

2. Install the local hook collector

Installer 
{{ install_command }}

The installer writes the hook script and env file under ~/.config/idle-rpg. It requires python3 to already be available in your shell.

3. Add the hook settings

Copy the snippet from the app into ~/.claude/settings.json or project-local .claude/settings.local.json. Merge it into the existing JSON instead of replacing the whole file.

4. Privacy model

  • No prompt text upload
  • No command text upload
  • No file path upload
  • Only turn summary counters and buckets are sent

Current behavior

  • SessionStart stores model information locally.
  • UserPromptSubmit starts a new turn.
  • PostToolUse and PostToolUseFailure update local summary counters.
  • Stop sends one turn_completed event to the server.

Settings File Guide

1
Choose where to install hooks

Use ~/.claude/settings.json for global use across projects, or .claude/settings.local.json for one repository only.

2
Open the current JSON file

If the file already contains settings, keep them and add the hooks section instead of overwriting unrelated keys.

3
Paste the snippet into the JSON file

The snippet is shown above and is ready to copy. After saving the file, Claude Code will call the local collector automatically.
