Claude Code Plugin

Install the Idle RPG plugin for Claude Code.

The plugin registers all hooks automatically and adds the /idle-rpg:rpg-status skill. Only privacy-safe summary metrics are sent — no prompt text, no file paths.

1. Create an installation

Open App, log in, and create a Claude Code installation. That gives you a dedicated API key.

2. Run the installer

Installer 
{{ install_command }}

Writes the plugin files to ~/.config/idle-rpg/plugin/ and your API key to ~/.config/idle-rpg/claude-code-hook.env. Requires python3.

3. Load the plugin

Start Claude Code with the plugin directory, or add it permanently to your settings.

One-time (per session) 
claude --plugin-dir {{ plugin_dir_default }}
.claude/settings.local.json 
{{ plugin_dir_snippet }}

After loading, hooks run automatically and /idle-rpg:rpg-status becomes available.

4. Privacy model

  • No prompt text upload
  • No command text upload
  • No file path upload
  • Only turn summary counters and buckets are sent

What the plugin provides

  • Hooks — SessionStart, UserPromptSubmit, PostToolUse, PostToolUseFailure, Stop registered automatically.
  • /idle-rpg:rpg-status — Shows your character level, stats, and active title in the console.

Hook behavior

  • SessionStart stores model information locally.
  • UserPromptSubmit starts a new turn.
  • PostToolUse and PostToolUseFailure update local summary counters.
  • Stop sends one turn_completed event to the server.
