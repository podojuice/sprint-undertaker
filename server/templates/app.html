Install Flow

Open Full Guide
1
Log in or register

The app stores your token locally so you can refresh character data and create installations.

2
Create one Claude Code installation

The server issues a dedicated API key for one local setup.

3
Run the installer command once

It writes the local Claude hook collector into ~/.config/idle-rpg and creates the hook env file automatically.

4
Add the hook settings snippet

Paste the generated JSON snippet into ~/.claude/settings.json or .claude/settings.local.json.

5
Use Claude Code normally

RPG tracking runs through Claude hooks without changing your usual prompt flow.
